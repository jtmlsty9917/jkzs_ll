## 项目简介

**懒懒健康助手** 是一款以陨石边牧「懒懒」为核心 IP 的移动端健康管理 PWA 应用。面向普通用户，提供喝水监督、锻炼记录、日记、待办提醒等功能，UI 风格简洁可爱。

## 工作指引

### 开发节奏
1. **小步迭代**：每次只做执行计划中的一个步骤，完成并验证后再继续
2. **每步可运行**：`npm run dev` 随时能看到界面变化
3. **先功能后打磨**：核心功能跑通再优化 UI 细节
4. **不跳步验证**：每步完成后手动测试已有功能

### 代码风格
- 使用 React 函数组件 + Hooks（不使用 class 组件）
- 组件文件使用 JSX 扩展名
- 每个组件一个文件，职责单一
- Hook 命名以 `use` 开头
- 优先使用 Tailwind CSS 原子类，复杂样式用 `index.css` 补充
- 中文注释 + 英文代码标识符

### 数据库操作
- 所有数据库操作通过 `src/db.js` 中定义的 Dexie 方法进行
- 不在组件中直接操作 IndexedDB
- 数据读写通过对应的 `useXxx` Hook 封装

### 路由
- 使用 Hash 路由（`createHashRouter`），兼容静态文件部署
- 路由表在 `src/App.jsx` 中定义

### UI 规范
- 设计基准 375px 宽，最大内容宽度 `max-w-md`
- 功能色严格使用设计规范中定义的值
- 所有交互元素最小 44x44px 触摸区域
- 底部导航固定，内容区需 `pb-20` 留白

## 关键约束

- **纯前端**：无后端服务，所有数据存于浏览器 IndexedDB
- **PWA 优先**：必须支持添加到手机桌面和离线访问
- **隐私优先**：不上传任何用户数据到服务器
- **移动端优先**：UI 针对手机屏幕优化，桌面端居中显示

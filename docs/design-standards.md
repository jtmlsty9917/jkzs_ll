# 懒懒健康助手 — 设计规范

> 版本：v1.0 | 创建日期：2026-06-24

---

## 一、设计理念

- **关键词**：简洁、可爱、温暖、有活力
- **情感基调**：像有一只真的边牧在身边陪伴你，软软的、偶尔调皮
- **视觉风格**：大面积留白 + 圆角卡片 + 柔和阴影 + emoji 点缀

## 四、字体

| 层级 | 大小 | 字重 | 用途 |
|------|------|------|------|
| H1 | `text-2xl` (24px) | `font-bold` | 页面标题 |
| H2 | `text-xl` (20px) | `font-semibold` | 区块标题 |
| H3 | `text-lg` (18px) | `font-medium` | 卡片标题 |

字体栈：`"PingFang SC", "Hiragino Sans GB", "Microsoft YaHei", sans-serif`（中文优先）

## 五、组件风格约定

### 卡片
- 白色背景 + 彩色顶部/左侧色条标记功能类型
- 柔和的 `shadow-md` + `rounded-2xl`
- 内边距 `p-4` 或 `p-5`

### 按钮
- 圆角 `rounded-full`（胶囊形）
- 主按钮：对应功能色填充 + 白色文字
- 次按钮：透明背景 + 功能色边框 + 功能色文字
- 点击态：`active:scale-95` 缩放反馈

### 输入框
- 圆角 `rounded-xl`
- 边框 `border-gray-200` + 聚焦时变功能色
- 内边距 `px-4 py-3`

### 日历格子
- 固定宽高比 1:1
- 日期数字在左上角
- 底部放运动图标（小尺寸 emoji）
- 背景色 = 喝水热力值对应蓝色

### 底部导航栏
- 固定底部，白色背景 + 顶部细阴影
- 4 个 Tab：🏠首页 / 💧喝水 / 📅日历 / ✅待办
- 当前 Tab 图标放大 + 文字变功能色

## 六、懒懒形象规范

- 类型：SVG 简笔风格陨石边牧
- 特征：黑白相间毛发、立耳、胸口白色倒 V、友善的表情
- 状态动画：
  - 默认：坐着，尾巴轻轻摇摆
  - 点击：跳一下 + 弹出表情气泡
  - 心情低落：耷拉耳朵、趴着
  - 心情好：耳朵竖起、星星眼

## 七、移动端适配

- 设计基准宽度：375px（iPhone SE）
- 最大内容宽度：`max-w-md` (448px)，居中显示
- 安全区：底部导航上方留 `pb-20`，顶部 `pt-4`
- 所有可点击元素最小 44x44px（符合触摸规范）

# 懒懒健康助手 — 项目需求文档

> 版本：v1.0 | 创建日期：2026-06-24 | 最后更新：2026-06-24

---

## 一、项目概述

### 1.1 项目名称
**懒懒健康助手** — 一款以陨石边牧「懒懒」为核心 IP 的移动端健康管理 PWA 应用。

### 1.2 目标用户
关注日常健康、需要喝水/锻炼提醒、喜欢可爱陪伴式体验的普通用户。

### 1.3 核心价值
- 用可爱的边牧形象降低健康管理的枯燥感
- 心情值机制提供正向激励
- 日历可视化让进步看得见
- 本地存储保护隐私，无需注册

---

## 二、功能需求

### F1 — 健康助手·懒懒（首页核心）

| 需求项 | 描述 |
|--------|------|
| F1.1 | 主页展示陨石边牧「懒懒」卡通形象 |
| F1.2 | 心情值计算：基于喝水/锻炼/日记/待办的完成情况动态计算（0-100分）|
| F1.3 | 心情分级：0-30（不开心）、31-60（一般）、61-85（开心）、86-100（超开心）|
| F1.4 | 点击懒懒弹出当前心情池随机表情 + 鼓励语 |
| F1.5 | 心情值实时随数据变化更新 |

### F2 — 监督喝水（蓝色系）

| 需求项 | 描述 |
|--------|------|
| F2.1 | 设置每日目标杯数 + 每杯毫升数 |
| F2.2 | 记录当日喝水杯数（0~20杯，支持 0.1 精度）|
| F2.3 | 当日进度条：已喝 vs 目标 |
| F2.4 | 月历热力图：用蓝色深浅表示每天喝水量 |
| F2.5 | 日历日期点击可看到当日详细喝水记录 |

### F3 — 监督锻炼（绿色系）

| 需求项 | 描述 |
|--------|------|
| F3.1 | 设置每周目标锻炼次数 |
| F3.2 | 记录锻炼：选择日期 + 运动类型 |
| F3.3 | 预设运动类型：跑步🏃 游泳🏊 健身🏋️ 骑行🚴 瑜伽🧘 球类⛹️ 羽毛球🏸 乒乓球🏓 网球🎾 散步🚶 跳舞💃 登山🏔️ |
| F3.4 | 月历中，日期格内显示运动图标 + 喝水热力背景 |

### F4 — 记录生活（粉色系）

| 需求项 | 描述 |
|--------|------|
| F4.1 | 每日上传照片（本地 base64 存储）|
| F4.2 | 每日写日记/感悟 |
| F4.3 | 日历双视图切换：视图A（运动图标+喝水热力）、视图B（照片缩略图）|
| F4.4 | 点击日期查看当天的完整记录 |

### F5 — 待办事项（暖橙色系）

| 需求项 | 描述 |
|--------|------|
| F5.1 | 添加待办：事项内容 + 截止时间 |
| F5.2 | 列表展示，可勾选完成/删除 |
| F5.3 | 到达设定时间 → 懒懒弹窗提醒（页面内动画 + 音效）|
| F5.4 | 逾期事项红色高亮标记 |

## 三、非功能需求

| 编号 | 需求 |
|------|------|
| NF1 | PWA — 支持添加到手机桌面（安卓/iOS）|
| NF2 | 离线可用 — Service Worker 缓存核心资源 |
| NF3 | 响应式设计 — 适配主流手机屏幕（375px-428px 宽）|
| NF4 | 数据安全 — 所有个人数据仅存于浏览器 IndexedDB |
| NF5 | 性能 — 首屏加载 < 2s，交互响应 < 100ms |
| NF6 | 可爱简洁的 UI 风格，白色底 + 各功能区活力色 |
